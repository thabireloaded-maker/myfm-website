# MY FM – I AM YOUTH

MY FM is a youth-driven online radio station based in Lesotho. The website allows listeners to stream live radio, explore show schedules, meet hosts, and interact with the MY FM community.

The platform promotes youth culture, music, and voices from Lesotho, South Africa, and the global youth community.

## Features
- Live radio streaming
- Show schedule
- Host profiles
- Listener chatroom
- Youth-focused music and discussions

## Technology
- HTML
- CSS
- GitHub Pages hosting
